課程說明

班級 : 資訊二甲 必修
課程 : 資料庫系統
教室 : 資電118
 教師 : 郭士煒 時間 : (一) 2,3,4
  1. 平時20% : 出席(5%) + 課堂作業(15%)
  2. 期中25% : 筆試
  3. 期末考25% : 筆試
  4. 期末專案30% : 專題報告

課程目標

1.瞭解資料庫管理系統之功能。
2.瞭解資料庫塑模和關連式資料庫之基礎與運用。
3.說明資料庫正規化的目的及方法。
4.運用SQL於查詢資料庫之資料。
